# Zac Lawless Brand Guidelines

## Logo Usage

- Primary logo: "Zac Lawless" text in bold sans-serif font
- Secondary mark: "ZL" monogram
- Minimum size: 100px width for logo, 40px for monogram
- Clear space: Minimum 20px around all sides

## Color Palette

### Primary Colors

- **Black**: `#000000` - Primary brand color for headers and emphasis
- **White**: `#FFFFFF` - For text on dark backgrounds and clean space

### Secondary Colors

- **Gray-50**: `#F9FAFB` - Light backgrounds
- **Gray-100**: `#F3F4F6` - Card backgrounds
- **Gray-600**: `#4B5563` - Body text
- **Gray-700**: `#374151` - Darker text
- **Gray-900**: `#111827` - Headings

### Accent Colors

- **Green-600**: `#16A34A` - For "Upcoming" badges and CTAs
- **Red-600**: `#DC2626` - For "Sold Out" indicators

## Typography

### Font Stack

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "Roboto", "Oxygen",
  "Ubuntu", "Cantarell", "Fira Sans", "Droid Sans", "Helvetica Neue", sans-serif;
```

### Type Scale

- **Display**: 48px-60px (Headings on hero)
- **H1**: 36px-48px (Page titles)
- **H2**: 30px-36px (Section headers)
- **H3**: 24px (Card titles)
- **Body**: 16px (Regular text)
- **Small**: 14px (Captions, labels)

## Musical Genre Tags

### Style

- Acoustic - Blue gradient
- Blue - Indigo gradient

- Funk - Purple gradient

## Social Media

### Profile Image

- Use high-quality artist photo
- Square format: 1:1 ratio
- Minimum 400x400px

### Cover Images

- Facebook: 820 x 312px
- Twitter: 1500 x 500px
- YouTube: 2560 x 1440px

### Content Style

- Authentic, personal tone
- Focus on music and live performances
- Behind-the-scenes content
- Engage with local community

## Photography

### Style Guide

- Natural lighting preferred
- Live performance shots
- Studio recordings
- Candid moments with guitar
- Venue atmospheres

### Format

- AVIF/WebP for web (optimized)
- High-res originals for press kit
- 16:9 for landscape, 1:1 for social

## Voice & Tone

### Brand Voice

- Authentic and personal
- Passionate about music
- Rooted in local community
- Welcoming and accessible

### Messaging

- Focus on 20+ years of experience
- Emphasize Pocono Mountains connection
- Highlight acoustic, intimate performances
- Promote upcoming shows and events

## Do's and Don'ts

### Do

✓ Use high-quality images
✓ Maintain consistent black/white branding
✓ Keep messaging authentic and personal
✓ Promote local venues and community
✓ Update show calendar regularly

### Don't

✗ Use low-resolution images
✗ Change brand colors
✗ Use generic stock photos
✗ Overly promotional language
✗ Neglect social media updates
